# LovacRaidButtons

## Introduction
**LovacRaidButtons** is a World of Warcraft addon designed to help raid leaders by eliminating the need to manually type common raid warnings. With customizable buttons and built-in utilities, you can send clear raid messages instantly.

## Features
- Add, edit, and remove custom raid warning buttons
- One-click raid warning messages
- Built-in pull timer
- Simple and lightweight UI

## Installation (via WowUp)
1. Open **WowUp**
2. Go to **Get Addons**
3. Click **Install from URL**
4. Paste the addon URL: `https://github.com/martinmatijevic/LovacRaidButtons`
5. Install and reload your game

## Usage
- Open the addon interface in-game
- Create buttons with predefined (or your own) raid warnings
- Click a button to instantly send the message as raid warning
- Button for start/cancel pull timer

## Intended Audience
- Raid Leaders
- Assistants
- Anyone responsible for raid coordination

## Requirements
- World of Warcraft (Retail)
- Raid leader or assist role
